# Как сделать так, чтобы я (AI) мог работать на сервере из Cursor

## В чём суть
Сейчас вы заходите на сервер по паролю. Я из Cursor могу запускать команды только на вашем Mac. Чтобы я мог выполнять команды на сервере, нужен вход по SSH-ключу: один раз добавить ключ с вашего Mac на сервер — дальше подключение будет без пароля, и я смогу подключаться сам.

## Что сделать один раз

1. Зайдите на сервер как обычно (в терминале: ssh debian@144.217.12.20, введите пароль).

2. В терминале НА СЕРВЕРЕ (когда видите приглашение debian@vps-39eb0606:~$) выполните одну строку ниже (скопируйте целиком, вставьте, Enter):

mkdir -p ~/.ssh && echo 'ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIPxwtYdkgwYjyQYLbu7sV7j1b/K4BUXWg9iKYqcTT5F5 alekseidoronin@mac-vps' >> ~/.ssh/authorized_keys && chmod 700 ~/.ssh && chmod 600 ~/.ssh/authorized_keys && echo Done.

3. Напишите мне здесь: «готово» или «сделал».

После этого я смогу из Cursor запускать на сервере команды вида:
  ssh debian@144.217.12.20 'ваша команда'
и вам не нужно будет ничего вводить — буду работать в «вашем» терминале на сервере автоматически.
